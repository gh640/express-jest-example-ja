# `express-jest-example-ja`

[Express](https://expressjs.com/) を使ったウェブアプリケーションのための自動テストのサンプルです。

## 使い方

パッケージをインストール:

```bash
npm install
```

テストを実行:

```bash
npm run test:unit

npm run test:integration

npm run test:e2e
```

## 利用パッケージ

### `depenencies`

- `express` 4.x

### `devDependencies`

- `jest` 29.x
- `jest-puppeteer` 10.x
- `start-server-and-test` 2.x
- `supertest` 6.x
